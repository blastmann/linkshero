Links Hero Privacy Policy

Last updated: 2026-02-22

简体中文

Links Hero（Chrome 扩展）用于从当前网页提取下载链接（如 HTTP/HTTPS、Magnet、Torrent），并可将链接复制、导出或推送到你配置的 Aria2 RPC 服务。

我们收集哪些数据

  • 我们不收集可识别个人身份的信息。
  • 我们不上传你的浏览历史或页面内容到开发者服务器。
  • 扩展配置（如 Aria2 RPC 地址、Token、下载目录、模板等）仅保存在浏览器本地 chrome.storage。

我们如何使用数据

  • 在你主动触发时扫描当前标签页中的下载链接。
  • 在本地去重、筛选、展示并执行复制/导出。
  • 按你的操作将选中任务发送到你配置的 Aria2 RPC 端点（例如 localhost:6800）。

数据共享与第三方

  • 我们不会将数据出售或共享给广告商。
  • 当你点击“推送到 Aria2”时，数据会发送到你指定的 Aria2 服务；该服务的数据处理行为由你自己的部署和配置决定。

数据保留与删除

  • 本地配置会保留在你的浏览器中，直到你手动清除扩展数据或卸载扩展。
  • 你可随时在扩展设置中修改配置。

权限说明（简要）

  • activeTab / scripting：在你触发时扫描当前页面链接。
  • tabs / webNavigation：定位标签与框架，确保链接提取完整。
  • storage：保存本地设置。
  • contextMenus / notifications：提供快捷入口与结果反馈。
  • clipboardRead / sidePanel / permissions：支持对应交互功能与权限管理。

联系我们

如有隐私问题，请通过 GitHub Issues 联系：
https://github.com/blastmann/linkshero/issues

English

Links Hero (Chrome extension) helps users extract download links from the current page (HTTP/HTTPS, Magnet, Torrent) and copy, export, or send them to a user-configured Aria2 RPC endpoint.

What data we collect

  • We do not collect personally identifiable information.
  • We do not upload browsing history or page content to developer servers.
  • Extension settings (such as Aria2 RPC URL, token, download directory, and templates) are stored locally in chrome.storage.

How we use data

  • Scan links on the active page only when the user triggers the action.
  • Perform local deduplication/filtering/display and copy/export actions.
  • Send selected tasks to the Aria2 RPC endpoint configured by the user (for example, localhost:6800).

Data sharing

  • We do not sell data or share it with advertisers.
  • When the user clicks "Push to Aria2", data is sent to the user-selected Aria2 service; handling at that endpoint depends on the user's own deployment.

Data retention and deletion

  • Local settings remain in the browser until the user clears extension data or uninstalls the extension.
  • Users can update settings at any time.

Contact

For privacy questions, please open an issue:
https://github.com/blastmann/linkshero/issues
