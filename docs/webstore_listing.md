## English

### Name
Links Hero

### Summary
Extract download links (Magnet/Torrent/HTTP) from web pages, support batch copy and push to Aria2 downloader.

### Description
Links Hero is a **productivity tool** designed for efficient download management. It helps developers and users batch extract download links from web pages, especially useful for **Open Source Mirrors (Linux ISOs)**, **Public Datasets**, and **Archived Files**.

Links Hero supports standard HTTP/HTTPS links, as well as **Magnet links** and **.torrent** files detection, perfectly integrating with Aria2 for automated workflows.

- One-click copy all selected links to clipboard.
- Export link list to a text file.

- **Minimal Permissions**: Scans only occur when you activate the extension.
- **Privacy First**: All processing happens locally in your browser. We do not collect browsing history or upload page content.

**How to Use:**
1. **Open**: Click the Links Hero icon in the toolbar; it automatically scans the page.
2. **Select**: Check the items you want (supports Select All).
3. **Action**: 
   - Click "Copy" to save links to clipboard.
   - Click "Export" to save as a file.
   - Click "Push to Aria2" to send tasks immediately (requires Aria2 configuration).
